
git init

git add filename or .

git commit -m "comment for this commit"

git checkout . (수정된 내용 초기화)

(원격 저장소 등록)

git remote add origin https://github.com/...

git remote -v

git push origin master

git pull origin master

git branch
	* : 현재 브랜치를 의미

git branch feature_x : 새로운 브랜치 생성

git checkout feature_x : 현재 브랜치 변경

git branch -b feature_x : 새로운 브랜치 생성 및 현 브랜치 변경

git checkout master : 현 브랜치 변경

git merge feature_x : 현 브랜치에 feature_x 를 적용함
